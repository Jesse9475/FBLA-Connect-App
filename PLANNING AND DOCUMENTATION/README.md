# Planning and Documentation Standards

This folder contains the living standards and evidence required for the FBLA
Connect app. These documents define how we plan, design, build, and document
the app using industry terminology and professional practices. They must be
referenced and updated whenever we add or change features.

## Standards Documents
- `planning_process.md`: planning artifacts, milestones, and decision logs.
- `classes_modules_components.md`: expert use of classes/modules/components.
- `mobile_architecture_patterns.md`: expert use of mobile architecture patterns.
- `documentation_and_copyright_compliance.md`: evidence-backed documentation
  and copyright compliance using legitimate sources.
- `data_handling_and_storage.md`: comprehensive and secure data handling.

## Usage Rule
- Every significant feature or change must update the relevant document(s).
